## 📝 Описание

Учебный проект, реализованный в рамках изучения работы с API и объектно-ориентированного программирования (ООП) на Python.

Программа получает данные о воздушных судах через внешние API и позволяет:
- получать список самолётов над выбранной страной
- фильтровать и сортировать их
- сохранять данные в файл (JSON / CSV)

Используются открытые API:
- OpenSky Network — данные о самолётах
- Nominatim (OpenStreetMap) — географические координаты стран

## 🚀 Быстрый старт

### Предварительные требования

- Python 3.9 или новее  
- Установленный pip  

---

### Установка

1. Установите Poetry:
```bash
pip install poetry
```

2. Установите Git:
https://git-scm.com/downloads

3. Клонируйте репозиторий:
```bash
git clone git@github.com:YOUR_USERNAME/airplane_tracker.git
```

```bash
cd airplane_tracker
```

4. Установите зависимости:
```bash
poetry install
```

---

### Возможности программы

После запуска пользователь может:

- Ввести название страны
- Получить список самолётов в её воздушном пространстве
- Отфильтровать:
  - по стране регистрации
  - по высоте полёта
- Отсортировать самолёты
- Получить топ N самолётов по высоте
- Сохранить данные в файл (JSON или CSV)

## ⚙️ Технические детали

### 📡 Работа с API

Реализована интеграция с двумя сервисами:

- Получение bounding box страны (Nominatim)
- Получение самолётов в указанной области (OpenSky)

Используется библиотека `requests`.

---

### 🧱 Архитектура проекта

Проект построен с соблюдением принципов ООП и SOLID:

- `BaseAPIClient` — абстрактный класс для работы с API  
- `AirplanesAPI` — реализация API-клиента  

- `Airplane` — модель самолёта  
  - валидация данных  
  - поддержка сравнения (по высоте и скорости)  
  - используется `__slots__`

- `BaseFileSaver` — абстрактный класс для работы с файлами  
- `JSONSaver`, `CSVSaver` — реализация сохранения данных  

- `utils.py` — вспомогательные функции:
  - фильтрация
  - сортировка
  - выбор формата сохранения

---

### 📂 Работа с файлами

Поддерживаются форматы:

- JSON
- CSV

Особенности:
- данные не перезаписываются
- отсутствуют дубликаты
- используется внутренний датасет
