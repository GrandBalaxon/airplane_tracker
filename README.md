## 📝 Описание

Airplane Tracker — это учебный проект, реализованный в рамках изучения работы с API и объектно-ориентированного программирования (ООП) на Python.

Программа интегрируется с внешними сервисами и предоставляет удобный интерфейс для получения, обработки и сохранения данных о воздушных судах.

### Основные возможности:
- получение списка самолётов над выбранной страной
- фильтрация по стране регистрации
- фильтрация по диапазону высот
- сортировка и выбор топ-N самолётов
- сохранение данных в файл (JSON / CSV)

Используемые API:
- OpenSky Network — данные о самолётах
- Nominatim (OpenStreetMap) — географические координаты стран

2. Установите Git:
https://git-scm.com/downloads

3. Клонируйте репозиторий:
```bash
git clone git@github.com:GrandBalaxon/airplane_tracker.git
```

```bash
cd airplane_tracker
```

4. Установите зависимости:
```bash
poetry install
```

## 🛠 Использование

### Запуск программы

Точкой входа является файл `main.py`.

```bash
python main.py
```

---

### Сценарий работы

После запуска программа:

1. Запрашивает страну.
2. Получает данные о самолётах через API.
3. Предлагает:
- сохранить данные (JSON / CSV).
- отфильтровать самолёты.
- выбрать диапазон высот.
- вывести топ-N по высоте.
4. Выводит результат в консоль.

## ⚙️ Технические детали

### 📡 Работа с API

Реализована интеграция с двумя сервисами:

- Получение bounding box страны (Nominatim)
- Получение самолётов в указанной области (OpenSky)

Используется библиотека `requests`.

---

### 🧱 Архитектура проекта

Проект построен с соблюдением принципов ООП и SOLID:

- `BaseAPIClient` — абстрактный класс для работы с API  
- `AirplanesAPI` — реализация API-клиента  

- `Airplane` — модель самолёта  
  - валидация данных  
  - поддержка сравнения (по высоте и скорости)  
  - используется `__slots__`

- `BaseFileSaver` — абстрактный класс для работы с файлами  
- `JSONSaver`, `CSVSaver` — реализация сохранения данных  

- `utils.py` — вспомогательные функции:
  - фильтрация
  - сортировка
  - выбор формата сохранения

---

### 📂 Работа с файлами

Поддерживаются форматы:

- JSON
- CSV

Особенности:
- данные не перезаписываются
- отсутствуют дубликаты
- используется внутренний датасет
